在爱尔兰留学有一下几个优势（这里主要讨论研究生项目，并主要和英国相比较）：

在申请过程的竞争压力会相对小一点。在几个英语为母语的国家中，爱尔兰属于比较小众的留学目的地，申请人数不多（但相对的，学校的体量也小）。对于背景稍显劣势的同学来说，爱尔兰学校的要求一般不会特别高（包括语言和绩点），而且也不涉及面试环节，按要求提供文书材料即可，认真准备申请到爱尔兰Top 2大学的概率也相当大。

项目时间短。爱尔兰的学制基本和英国相似，研究生的1年时间即可毕业。全年的课程设计大体是前2个学期（9月-12月/1月-5月）授课，最后1学期（6月-8月）写论文/实习，比较紧凑，适合以较短的时间获得一个找工作的跳板学历。

英语母语，不需要额外学习二外。和其他欧陆的留学目的地相比，爱尔兰是唯一一个以英语为母语的国家，项目也都是英语授课。在项目时间短的情况下，无需学习二外一方面可以更快融入当地的生活，一方面也可以把经历更好的放在学习/找工作上。

费用相对低廉。学费的方面，我觉得整体相比英国的学校会便宜一些。生活成本而言，从我自身的体验看，首都都柏林的房租和物价水平都直逼伦敦，都柏林之外的地方生活成本就比较合理。网络上有大量针对都柏林房价的讨论，背后主要的原因是大量的公司入驻带来了人口涌入，但是在房屋供给侧，一方面没有像伦敦一样成熟的租房市场，一方面新房屋开发缓慢又以容积率的house为主，导致租房价格高企。

爱尔兰留学的劣势主要集中在：

学校在国内的知名度和认知度比较低。相比其他留学产业更发达的国家，爱尔兰学校在国内的认可度会稍微低一些，我不确定这会不会在回国找工作的过程中带来一些困难。我身边不少的同学也是以爱尔兰的学历得到了不错的工作机会，所以我感觉可能更多还是看个人的能力吧。

欧洲游玩。爱尔兰虽然属于欧盟，但是不在申根签的体系之内，所以去欧陆国家都需要再额外办理签证。如果你想借着这一年的时间游遍欧洲的话，在爱尔兰虽然地理距离不远，但是出行需要办理的手续还是相当复杂的。而且以学生签的身份，哪怕是去英国的话也需要再办英签，会给出行增加一些不便。

课程设置。我自己读的是圣三一学院(Trinity College Dublin)的MSc in Computer Science项目。我个人的感觉是项目比较偏重理论，虽然开设在计算机学院之下，但是涉及数学的内容相当多，实操比较少，对编程能力没有很大提高。

关于工作签证和工作机会：

研究生毕业后即可获得2年签证(Stamp 1G)找工作(具体细节可以查看官网)。这个签证在毕业之后申请，一年一续，最长2年，旨在提高爱尔兰高等教育的吸引力以及非EU学生在爱尔兰工作的竞争力。一般大型的雇主都会sponsor Stamp 1签证，即正式的工签；但对于暂时不能/不愿sponsor工签的雇主而言，Stamp 1G保证了你可以以合法的身份入职。如果你在公司期间表现较好，那么相信雇主也是愿意在签证期结束之后帮助你申请正式的工签。不过，也存在一些雇主只是借此剥削graduate，用后即扔的情况，这个需要你在面试的时候多留个心眼，或者借助LinkedIn等渠道找到已经入职的同事，了解清楚情况。

工资水平。从人均GDP的角度来看，似乎爱尔兰的排名还是相当高的，但从真实的人均工资水平而言，这个数字绝不如人均GDP看起来那么美妙（知乎有一个帖子讨论为什么爱尔兰的GDP水平不能够很好地衡量这个国家的真实经济水平，值得一读）。在没有工作经验的情况下，金融/商科毕业的同学起薪大概在30k上下，做software enginee的话大约在35k-40k这样。爱尔兰的个人所得税税率也是蛮高的，具体每个月到手能拿多少钱可以在Salary calculator上算一下。

工作机会。如果你是奔着“欧洲硅谷”的名头，想尽大FLAG firm所以才来的爱尔兰，那我个人的建议不是特别值得。因为在都柏林设置的岗位中虽然有开发，但是数量肯定没有美国那么多，竞争也挺激烈的，收入水平那也和美国没有办法比（我自己感觉我身边留下的同学虽然不少，但是真的进大Tech公司的人数还是很有限的）。但是，爱尔兰software engineer的人才缺口还是真实存在的，所以如果不挑公司的话，那我相信只要努努力多刷题多练习，在当地找到工作不是一件特别困难的事情。

如果有移民国外的打算，那爱尔兰算是一个不错的选择。假如你的工作属于紧缺技能职业，工资达到一定水平（一般software engineer都是），那么以Stamp1的身份工作2年之后你就有申请爱尔兰永居的机会，进而可以申请移民。有了永居之后不需要雇主sponsor工作签证，在职业方向的选择上也可以更自由一些。
